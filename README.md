# UR10e + Robotiq Hand-E URDF Builder

Build a combined URDF for the Universal Robots UR10e manipulator with Robotiq Hand-E gripper.
Generates URDF files with default or calibrated parameters and UR robot descriptions

To build run `build.sh` and find the output in the `out` folder.

Based on:
* https://github.com/macmacal/robotiq_hande_description.git
* https://github.com/UniversalRobots/Universal_Robots_ROS2_Description.git

Prerequisites:
* Docker
